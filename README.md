# 🚀 GFG 160 Days Coding Challenge

This repository documents my **160 Days Coding Challenge** based on **GeeksforGeeks (GFG)** problem sets. The goal of this challenge is to strengthen **Data Structures & Algorithms**, improve **problem-solving skills**, and maintain **daily coding consistency**.

I will be solving **one problem every day** and pushing the solution to GitHub along with proper explanations and optimized code.
